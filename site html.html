O que é uma Page Aplication?

Single Page Application SPA Single Page Application é uma aplicação ou web site que interage com o usuário de forma dinâmica sem que precise recarregar novos arquivos. Todos os arquivos necessários para o site ou aplicação funcionar são baixados apenas uma vez e depois são iterados sem afetar a experiência do usuário, deixando uma navegação mais fluída e natural. Essa navegação mais fluida de sites trás para a web a mesma experiência que temos em aplicativos desktop.
